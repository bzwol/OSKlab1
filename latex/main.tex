\section{Opis realizowanego zadania}
Utworzono kalkulator z zegarem. Kalkulator rozwiązuje wpisywane równania wprowadzane za pomocą przycisków w okienku programu. Zaimplementowano dwa style zegarów -- analogowy i cyfrowy.
Program zrealizowano za pomocą WindowsForms w języku C\#. Program można uruchomić tylko na komputerach z systemem operacyjnym Windows.

\section{Metody implementacji}
Obliczanie równania wprowadzonego do kalkulatora realizowane jest poprzez konwersje równania z postaci infiksowej na odwrotną notację polską. Tak przetworzone równanie jest następnie rozwiązywane, a wynik obliczeń wyświetlany jest w oknie w którym wprowadzano równanie.

\begin{lstlisting}[style=csharp]
private string Convert_To_RPN(object sender, EventArgs e)
{
    Label label = sender as Label;
    Stack<char> stack = new Stack<char>();
    string expression = label.Text;
    string output = null;

    for (int i = 0; i < expression.Length; i++)
    {
        char c = expression[i];
        if (char.IsDigit(c) || c == ',')
        {
            if (c == ',')
            {
                output += ',';
            } else
            output += c;
        }
        else if (c == '+' || c == '-' || c == '*' || c == '/')
        {
            output += ' ';
            while(stack.Count() > 0 && (stack.Peek() == '*' || stack.Peek() == '/'))
            {
                output += " " + stack.Pop() + " ";
            }
            stack.Push(c);
        }
        else if (c == '(')
        {
            stack.Push(c);
        }
        else if (c == ')')
        {
            while(stack.Peek() != '(')
            {
                output += " " + stack.Pop() + " ";
            }
            if(!(stack.Count() > 0))
            {
                MessageBox.Show("Nie ma zamykajacego nawiasu!");
                this.valid_RPN = false;
            }
            else
            {
                stack.Pop();
                this.valid_RPN = true;
            }
        }
    }

    while (stack.Count > 0)
    {
        output += " " + stack.Pop();
    }
    return output;
}
\end{lstlisting}

Zegar analogowy posiada trzy wskazówski wskazujące na godzinę, minutę i sekundę aktualnego czasu oraz znaczniki pomagające odczytać dokładną godzinę. Zegar cyfrowy składa się z czterech siedmiosegmentowych wyświetlaczy wyświetlających godzinę i minutę. Pomiędzy wyświetlaczami znajduje się dwukropek pulsujący co sekundę.

Za pomocą obiektu \emph{Timer} zapisywano informacje dotyczące aktualnego czasu, które następne wykorzystano do określenia punktów początku i końca liń reprezentujących wskazówki.

W celu umożliwienia użytkownikowi zmiany wyglądu programu oraz typu zegara zaimplementowano menu, w którym znajdują się odpowiednie opcje. Zaimplementowano motywy: domyślny, ciemny i zielony.
