\section{Opis realizowanego zadania}
Utworzono kalkulator z zegarem. Kalkulator rozwiązuje wpisywane równania wprowadzane za pomocą przycisków w okienku programu. Zaimplementowano dwa style zegarów -- analogowy i cyfrowy.
Program zrealizowano za pomocą WindowsForms w języku C\#. Program można uruchomić tylko na komputerach z systemem operacyjnym Windows.

\subsection{Kalkulator}
Obliczanie równania wprowadzonego do kalkulatora realizowane jest poprzez konwersje równania z postaci infiksowej na odwrotną notację polską, i następne rozwiązanie równania zapisanego w takiej postaci. Postąpiono tak w celu umożliwienia realizacji rozwiązywania wielooperatorowych równań.

private string Convert_To_RPN(object sender, EventArgs e)
{
    Label label = sender as Label;
    Stack<char> stack = new Stack<char>();
    string expression = label.Text;
    List<string> output = new List<string>();
    StringBuilder number = new StringBuilder();

    for (int i = 0; i < expression.Length; i++)
    {
        char c = expression[i];

        if (char.IsDigit(c) || c == ',')
        {
            number.Append(c);
        }
        else
        {
            if (number.Length > 0)
            {
                output.Add(number.ToString());
                number.Clear();
            }

            if (c == '+' || c == '-' || c == '*' || c == '/')
            {
                while (stack.Count > 0 && stack.Peek() != '(' &&
                        GetPrecedence(stack.Peek()) >= GetPrecedence(c))
                {
                    output.Add(stack.Pop().ToString());
                }
                stack.Push(c);
            }
            else if (c == '(')
            {
                stack.Push(c);
            }
            else if (c == ')')
            {
                while (stack.Count > 0 && stack.Peek() != '(')
                {
                    output.Add(stack.Pop().ToString());
                }
                if (stack.Count == 0)
                {
                    MessageBox.Show("Brakujący nawias otwierający!");
                    this.valid_RPN = false;
                    return "";
                }
                stack.Pop();
            }
            else if (c != ' ')
            {
                MessageBox.Show($"Nieznany znak: {c}");
                this.valid_RPN = false;
                return "";
            }
        }
    }

Algorytm konwersji zapisu równania polega na zapisywaniu operatorów na stosie, z którego są następnie zdejmowane wedle kolejności wykonywania działań i występujących liczb w równaniu.
Tak przetworzone równanie jest następnie rozwiązywane, a wynik obliczeń wyświetlany jest w oknie w którym wprowadzano równanie.

private float Solve_RPN(string rpn)
{
    float result = 0;
    Stack<float> stack = new Stack<float>();
    string number = null;
    if (rpn == null) return 0;
    for (int i = 0; i < rpn.Length; i++)
    {
        char c = rpn[i];
        if (char.IsDigit(c) || c == ',')
        {
            number += c;
        }
        else
        {
            if (!string.IsNullOrEmpty(number))
            {
                stack.Push(float.Parse(number));
                number = null;
            }
            if ((c == '+' || c == '-' || c == '*' || c == '/') & stack.Count > 0)
            {
                float a = stack.Pop();
                float b = stack.Pop();
                switch (c)
                {
                    case '+':
                        result = b + a;
                        break;
                    case '-':
                        result = b - a;
                        break;
                    case '*':
                        result = b * a;
                        break;
                    case '/':
                        result = b / a;
                        break;
                }
                stack.Push(result);
            }
        }
    }
    if (stack.Count > 0) return stack.Pop();
    else return float.Parse(number);
}

\subsection{Interfejs}
Wprowadzanie informacji do kalkulatora obsługiwane jest poprzez przyciski widoczne pod zegarem. Widnieją tam przyciski do wprowadzenia cyfr, przecinka, operacji dodawania, odejmowania, mnożenia, dzielenia i nawiasów. Istnieje również przycisk "C", który usuwa zawartość okna wejściowego. Zaimplementowano również wsparcie klawiatury, gdzie po za obsługą wciskania klawiszy odpowiadającym przyciskom pokazanym na ekranie, klawisz Enter również pełni funkcję znaku "=", a klawisz Backspace umożliwia usuwanie znaków jeden po jednym z okna wejściowego.

\subsection{Zegary}
Zegar analogowy posiada trzy wskazówski wskazujące na godzinę, minutę i sekundę aktualnego czasu oraz znaczniki pomagające odczytać dokładną godzinę. Zegar rysowany jest poprzez wyznaczanie kąta obrotu poszczególnych wskazówek dla aktualnie zmierzonego czasu. Zegar cyfrowy składa się z czterech siedmiosegmentowych wyświetlaczy wyświetlających godzinę i minutę. Każdy segment danego wyświetlacza posiada dwa stany: włączony i wyłączony. Na podstawie aktualnego czasu zapalane są dane segmenty wedle zapisanych trwale kombinacji. Pomiędzy wyświetlaczami znajduje się dwukropek pulsujący co sekundę. 

Aktualny czas uzyskiwano za pomocą obiektu \emph{Timer}.
